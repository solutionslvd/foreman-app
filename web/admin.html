Stripe Payment Setup
Complete these steps to start accepting payments from your customers.
1

Create a Stripe Account

Go to stripe.com and sign up. Complete identity verification. Canadian businesses select Canada as country.

2

Get Your API Keys

Stripe Dashboard → Developers → API Keys. Copy your Publishable Key (pk_live_…) and Secret Key (sk_live_…).

3

Add Keys to Render Environment

Render dashboard → foreman-app → Environment → Add variables:

STRIPE_PUBLISHABLE_KEY=pk_live_your_key_here
STRIPE_SECRET_KEY=sk_live_your_key_here
4

Set Up Webhooks

Stripe Dashboard → Developers → Webhooks → Add endpoint:

https://foreman-app.onrender.com/api/billing/webhook

Events: payment_intent.succeeded, customer.subscription.updated, customer.subscription.deleted, invoice.payment_failed

5

Create Products in Stripe

Stripe Dashboard → Products → Add Product for each plan. Copy Price IDs:

STRIPE_PRICE_STARTER=price_xxx
STRIPE_PRICE_PROFESSIONAL=price_xxx
STRIPE_PRICE_BUSINESS=price_xxx
STRIPE_PRICE_ENTERPRISE=price_xxx
6

Configure Bank Account

Stripe Dashboard → Settings → Payouts → Add your Canadian bank account. Stripe deposits every 2 business days.

7

Enter Keys & Save
