📲 PWA Deployment (Current)

Foreman is currently deployed as a Progressive Web App (PWA). Users can install it on their phone from the browser. For native app store listings, follow these steps:

1

Wrap with Capacitor (iOS/Android)

Run: npm install @capacitor/core @capacitor/cli && npx cap init to wrap the web app as a native app.

2

Apple Developer Account

$99 USD/year. Required to publish on App Store. Register at developer.apple.com.

3

Google Play Developer Account

$25 USD one-time fee. Register at play.google.com/console.

4

App Store Screenshots

Create 5-10 screenshots showing key features. Use iPhone 6.7" and Android phone sizes. Tools: Figma, Canva, or AppMockUp.com.
